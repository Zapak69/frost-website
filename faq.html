Frequently Asked Questions

Last updated: July 5, 2026

General

What is Frost?

Frost is a free Minecraft client built around speed, customization, and simplicity. It ships with 60+ built-in mods, a friend list, custom capes, a mod browser, and both Vulkan and OpenGL rendering — no manual file installs or mod manager required.

Is Frost really free?

Yes. Frost is, and always will be, free to download and use. During Early Access, downloads are gated behind an access code, but the client itself doesn't cost anything.

What is the Early Access Code for?

Frost is currently in a private Early Access phase, so downloads are limited to people who have a code. Codes are handed out by the team and through the community — check our Discord server for the latest on how to get one.

Will using Frost get me banned on servers?

Frost is a client-side utility mod, similar to other Minecraft clients. Some servers restrict certain mods or clients in their rules, so always check a server's policy before joining. Sharing or leaking Early Access builds is against our own terms and can get your access revoked — see the notice shown when you redeem a code.

Which Minecraft versions does Frost support?

Frost currently targets Minecraft 1.21.x. Check the download section on the homepage for the exact version supported by the latest build.

I lost my code or it stopped working — what do I do?

Reach out on our Discord server and we'll help you sort it out.

Vulkan vs. OpenGL

What's the difference between Vulkan and OpenGL?

Both are graphics renderers Frost can run on. Vulkan is the newer, lower-overhead option and gives the biggest FPS boost on most modern systems. OpenGL is the older, more broadly compatible option — it runs on more hardware and drivers, just with less of a performance gain.

Which one should I download?

Start with Vulkan for the best performance. If it fails to launch, crashes, or your GPU driver doesn't support it, come back and grab the OpenGL version instead — your Early Access code lets you download each renderer once.

Installation

Frost is distributed as a .mrpack modpack file through Modrinth. Installing it takes two steps once you have your download.

1

Install the Modrinth App

Frost installs through the Modrinth App — download and install it first if you don't already have it.

2

Redeem your Early Access code

Head back to the homepage, enter your code, and download the Vulkan or OpenGL build.

3

Open the downloaded .mrpack file

That's it — just double-click the downloaded file. The Modrinth App picks it up automatically and installs Frost Client for you, no manual setup required.

4

Launch from the Modrinth App

Once the install finishes, you'll find Frost listed as an instance in the Modrinth App — select it and press Play.

Don't own Minecraft? The official Modrinth App requires a premium (Microsoft-authenticated) Minecraft account. If you want to play on a cracked/offline account, use AstralRinth instead — a fork of the Modrinth App with offline account support. Install it the same way as the Modrinth App above, then follow the same steps: open the downloaded .mrpack file and Frost installs itself inside AstralRinth too.
